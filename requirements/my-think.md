跟后端同事在线会议讨论:

我们主要有 3 部分，第一部分是 登录，第二部分是 mcp，第三部分是 会话。

登录现在基本满足了，还缺少 刷新令牌。

mcp 包括:

- mcp 注册 (可以允许用户将 json 内容粘贴过来，因为要填的东西太多了)
- mcp 认证 (可以允许用户将 json 内容粘贴过来，因为要填的东西太多了。现在认证类型只做简单的 API_KEY)
- mcp 同步 (需要在认证成功后默认执行一次)
- mcp 管理等功能，我就不逐个列出来了

会话 包括:

- 可以获取会话列表
- 可以进入到具体的会话查看消息历史
- 也可以创建一个新的会话 (session)
- 在会话里面才能创建 task
- 在会话里面才能管理 task
- 会话管理、task 管理等功能我就不逐个列出来了
- 后端同事说借鉴的是 ChatGPT 聊天思想，可以创建多个聊天，在聊天里面有不同的内容或任务
